# Spiller Seire Kamper Vinn%

Siste kamper

Ingen avsluttede kamper ennå. Spill en kamp og avslutt den for å se statistikk her.
Slik fikser du det: Gå til Firebase Console → Firestore → Rules, og legg til tilgang for history: 
match /history/{doc} {
  allow read, write: if request.auth != null;
}
